## ğŸ“¦ Installation

### Requirements
- Minecraft **1.21.5**
- Fabric Loader **0.16.13+**
- Fabric API
- Java **21+**

### Steps
1. Install [Fabric Loader](https://fabricmc.net/use/installer/)
2. Download the latest Arc release
3. Place `arc-x.x.x.jar` in your `.minecraft/mods` folder
4. Launch Minecraft with the Fabric profile
